Minetest Game mod: concreted
=======================

This mod adds new concrete blocks for every dye color in the default game.
The textures used in the mod were chosen in order to get as close as possible to the default colors of the game.
Each concrete color has stair, slab, and wall variants, too.
The main purpose of this mod is to add to the default game a new block that perfectly matches the game's default appearance and mechanics.
